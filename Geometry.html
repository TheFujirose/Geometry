The Point

The Point is a zero-dimentional object meaning that it has no width, length or height.
A Point can exist in any dimention and functions as the location within any dimention. This is specified using an n-tuple
Points are discribed as a dot but take into mind that a point has no dimentions.



Points come in many forms:
  • Midpoint
    Points that devide a line segment into two segments of equal length; The middle of the segment.

  • Endpoint
    Points at the boundary of a line segment.

  • Antipodal Point

  • Isotropic Point

  • Diagonal Point

  • Collinear Points
    Points that exist on the same line

  • Non-Collinear Points
    Points that don't exist on the same line

  • Coplanar Points
    Points that exist on the same     plane

  • Non-Coplanar Points
    Points that don't exist on the same     plane
